Flappy Bird - AI

Description

  • Siddhesh Gunjal

Objective of the project was to make a flappy bird Agent game which can learn to play the game itself and break my high score (which is 93).

Conclusion: After the Agent was trained properly it was able to beat my high score and reach the highest score of 268.

Technology

  • Python
  • TensorFlow
  • Reinforcement Learning
  • Deep Q-learning Networks
  • PyGame
